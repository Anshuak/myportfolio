# Personal Portfolio Website

This is a high-end, modern, animated personal portfolio web application built with React, Vite, Tailwind CSS, and Framer Motion.

## Features

- **Gaming-inspired Dark Theme:** A sleek, futuristic design with neon accents.
- **Smooth Animations:** Built with Framer Motion for a fluid user experience.
- **Responsive Design:** Mobile-first and looks great on all devices.
- **Reusable Components:** Clean, well-structured code.
- **Unique Gaming Section:** Showcasing gaming achievements and setup.

## Tech Stack

- **React (v19)**
- **Vite**
- **Tailwind CSS (v3)**
- **Framer Motion**
- **React Router**
- **React Icons**

## Setup and Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/your-portfolio.git
    cd your-portfolio
    ```

2.  **Install dependencies:**
    Make sure you have Node.js and npm installed.
    ```bash
    npm install
    ```

## Running the Application

To start the development server, run the following command:

```bash
npm run dev
```

This will start the application on `http://localhost:5173` (or the next available port).

## Building for Production

To create a production build, run:

```bash
npm run build
```

The optimized static files will be located in the `dist` directory.

## Customization

- **Project Data:** Edit the `src/data/projects.js` file to add your own projects.
- **Skill Data:** Edit the `src/data/skills.js` file to update your skills and proficiency levels.
- **Personal Information:** Update the content in the components inside the `src/components` directory (e.g., `About.jsx`, `GamingSection.jsx`, `Contact.jsx`) with your own details.
- **Styling:** Modify colors, fonts, and other styles in `tailwind.config.js` and `src/index.css`.
